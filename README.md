# An-lisis-de-una-empresa-de-telecomunicaciones
reporte para entender cómo los clientes usan realmente los servicios móviles (llamadas y mensajes).  El objetivo de la empresa es identificar patrones de uso, detectar comportamientos atípicos y comprender qué segmentos de clientes muestran necesidades diferenciadas, con el fin de optimizar la oferta comercial y mejorar la experiencia del usuario.
